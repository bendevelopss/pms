*********** Please run the software with the following requirements ******************

1. Compatible with PHP5 and lower versions.
2. MySQL version must be 5.6 or lower versions.
2. Choose the correct destination of the software to be run. (e.g. C:\xampp\htdocs\)


******************* Steps by step on how to run the sofware ********************
1. Run "setup.exe".
2. Click "Install" to install the software. Wait until the installation is done.
3. Create new database named "pms"(disregard the quotations) using phpmyadmin in your browser (localhost/phpmyadmin).
4. Import the sql file named pms.sql inside the folder named "Database".
5. Now access the url for the software using your browser. (e.g. localhost\htdocs\pms).
6. Use 'admin' as username and 'admin' as the password to login as an administrator;
7. Please check the documentation for more information of usage.
